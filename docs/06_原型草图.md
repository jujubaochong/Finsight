## 页面 1：首页 / 搜索页

### 布局

```
┌──────────────────────────────────────────────────────────┐
│  FinSight  [自选股]                           [关于]     │
├──────────────────────────────────────────────────────────┤
│                                                          │
│                                                          │
│                    AI 驱动的智能投研助手                    │
│                                                          │
│         ┌─────────────────────────────────────┐          │
│         │  🔍  输入股票代码或名称...           │          │
│         └─────────────────────────────────────┘          │
│                                                          │
│         试试这些：                                        │
│         [贵州茅台] [宁德时代] [比亚迪] [药明康德]           │
│                                                          │
│         ── 最近浏览 ──                                   │
│         平安银行 · 000001    刚刚                         │
│         招商银行 · 600036    2小时前                      │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

### 关键元素
- **搜索框**：居中，自动聚焦。placeholder："输入股票代码或名称，如 平安银行 / 000001 / payh"
- **热门搜索**：静态推荐3-5只热门股票，点击直接跳转（给新用户一个"立即体验"的入口）
- **最近浏览**：从 localStorage 读取，展示最近3条浏览记录

### 交互说明
- 输入 ≥1 字符 → debounce 300ms → 下拉展示搜索建议（最多5条）
- 搜索建议每行：代码 + 名称 + 市场标签（SZ蓝色/SH红色）
- 键盘 ↑↓ 选择，回车跳转
- 点击建议项或按回车 → 跳转 `/stock/{code}`

### 滚动行为
- 页面从上到下依次排列，不折叠（用户需要一次性浏览全貌）
- AI 快速分析和风险提示在首屏必须可见（最重要的信息不做隐藏）
- 财务图表在第二屏（需要一点滚动，但核心指标卡已把关键数字前置了）

### 核心交互
- 点击指标卡 → 跳转到财务图表的对应指标位置
- 切换"年度/季度"视图 → 图表和指标卡联动刷新
- 点击"展开完整分析" → AI 分析区域展开，展示亮点+风险详情
- 公告列表每条可展开看 AI 摘要（默认折叠，减少页面长度）

## 交互流程图（核心路径）

```
用户访问首页
    │
    ├── 搜索框输入"平安银行" → 下拉建议出现
    │       │
    │       └── 点击建议项 / 回车
    │              │
    │              ▼
    │        股票详情页 /stock/000001
    │              │
    │              ├── 浏览 AI 一句话总结 + 指标卡 + 图表 + 风险
    │              │
    │              ├── [加入自选] → 添加到自选列表
    │              │       │
    │              │       └── 后续：收到异动通知 → 回到详情页
    │              │
    │              └── [生成完整报告] → 跳转报告页 /stock/000001/report
    │                     │
    │                     ├── 等待生成（进度条）
    │                     │
    │                     ├── 查看报告（左右分栏阅读）
    │                     │
    │                     └── [导出 Word/PDF] → 下载文件
    │
    └── 点击 [自选股] → 自选股列表页 /watchlist
           │
           └── 点击任意股票 → 回到股票详情页
```
